# Сказочный квест: *Легенды леса*

Учебный курсовой проект: визуальный квест на PyQt6 про путешествие героя по сказочному лесу, встречи с Лешим, Водяным, Бабой Ягой и финальную битву с колдуном.

---

## Ключевые возможности 🛠️

- Нелинейный сюжет с влиянием **кармы** на финальную битву.
- Несколько мини-игр: бой с чёртиком по кликам, сортировка корней, «три в ряд», загадки.
- Система инвентаря: оружие, зелья, золота.
- Атмосферное оформление: фоны, спрайты персонажей, фоновая музыка.
- Локальное сохранение ключевых параметров (имя героя, карма, ресурсы).

---

## Техническая реализация (архитектура)

- Язык: **Python 3.9**
- GUI: **PyQt6**
- Основные компоненты:
  - `main.py` — точка входа, создание `QApplication`, запуск окна `FairyQuestGame`.
  - Класс `FairyQuestGame` — главное окно, навигация по сценам, обработка выбора игрока.
  - Класс `GameSession` — состояние игры (имя игрока, карма, здоровье, инвентарь, язык, тема).
  - Класс `Inventory` — золото, артефакты, оружие, зелья, картинки для иконок.
- Ресурсы:
  - Папка `images/` — фоны, спрайты персонажей, иконки оружия и зелий.
  - Папка `music/` — фоновая музыка и звуки.
- Музыка подключается через `QMediaPlayer` и `QAudioOutput`.

## Как запустить проект из исходников

### 1. Клонировать репозиторий

```bash
git clone https://github.com/Williams200630/FairyQuestGame.git
cd FairyQuestGame
```

### 2. Установить зависимости

Рекомендуется использовать виртуальное окружение:

```bash
python -m venv venv
venv\Scripts\activate

pip install -r requirements.txt
```

Если файла `requirements.txt` пока нет, можно установить напрямую:

```bash
pip install PyQt6
```

### 3. Запустить игру

```bash
python main.py
```

---

## Как собрать .exe (Windows)

### 1. Установить PyInstaller

```bash
pip install pyinstaller
```

### 2. Собрать исполняемый файл

Находясь в корне проекта (`Cursach`):

```bash
py -m PyInstaller --onefile --windowed --add-data "images;images" --add-data "music;music" main.py
```

Готовый файл будет лежать по пути:

```text
dist/main.exe
```
